# Wonders Pictures — Cinematic Videography Portfolio

A premium Next.js 14 videography portfolio and booking website.

## Tech Stack
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS** (custom cinematic design tokens)
- **Framer Motion** (scroll-triggered animations)

## Design
- **Display font**: Cormorant Garamond (editorial, cinematic)
- **Body font**: DM Sans
- **Palette**: Obsidian (#080808) · Ivory (#F0EBE3) · Gold (#C4A265)
- **Signature element**: Film strip ticker running between sections

---

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

---

## Pages
| Route | Description |
|---|---|
| `/` | Homepage with hero, films, reel, services, process, testimonials |
| `/portfolio` | Filterable film grid |
| `/about` | Director story, philosophy, awards |
| `/services` | Full service listings with pricing |
| `/contact` | Booking form + WhatsApp CTA |
